# Proyecto de Automatización Correos

## Comandos para colocar en terminal
>[!IMPORTANT]
>Comando para prender el puerto
>```bash
>npm start
>```
### Otros comandos
```bash
npm run build
npm run
npm install 
npm i '...'
```
---

>[!IMPORTANT]
> Para que funcione el aplicativos se debe descarcar todas las dependencias con este comando.
>
> ```bash
> npm install .
> ```
